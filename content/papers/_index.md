### Occupational asthma research papers 

We have collated over 5000 research papers related to occupational asthma, over 2,500 with abstracts. These can be downloaded in Medline format for use with reference management software, such as Cite while you write and Reference Manager.

Many of these have been converted from a Reference Manager Database which was largely hand entered. Please excuse any typing errors.
